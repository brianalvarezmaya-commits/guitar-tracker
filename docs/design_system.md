# Sistema de Diseño

## Paleta de Colores

- Primario: #7C3AED (Morado)
- Secundario: #F59E0B (Naranja)
- Fondo: #F3F4F6 (Gris claro)
- Texto: #1F2937 (Gris oscuro)
- Error: #EF4444 (Rojo)

## Tipografía

- Fuente principal: 'Roboto', sans-serif
- Tamaño base: 16px

## Componentes

### Botones
- Bordes redondeados (8px)
- Color primario
- Hover con sombra

### Tarjetas
- Fondo blanco
- Sombra ligera
- Bordes redondeados
- Padding de 16px

Sistema de 8px (8, 16, 24, 32)
